# Documentation Index

This directory contains internal documentation for `@lean-poker/boardgame.io` maintainers and contributors.

## Structure

- `breaking-changes/` -- Documented breaking changes by version
- `architecture/` -- Long-term architectural discussions and design decisions

## Guidelines

- Use Markdown format for all documents
- Include code examples with language tags
- Link to related issues and PRs for traceability
- Keep agent-readable: clear headings, before/after comparisons
